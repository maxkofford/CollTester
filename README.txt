# Demo_Project
The September demo project.

Folder organization:
====================

shared_assets - contains assets that don't have their own git repository and that are symbolically linked to from the Unity project Assets folder. Check the readme inside for details on its organization.

submodules - contains git submodules used in this repository.

Raw_Files - contains source files and assets for external tools.

Unity - the project folder for this repository. Currently we support only a single Unity project per repository.
